\# Helmet Detection AI 



A computer vision object detection project to identify:

\- \*\*rider\*\*

\- \*\*with helmet\*\*

\- \*\*without helmet\*\*



\## Why this matters

Helmet compliance saves lives, but manual monitoring is slow and inconsistent.

This project explores automated helmet compliance detection using object detection.



\## Dataset

\- Total images: ~15,000+

\- Annotated: ~9,000+

\- Split: 70/20/10

\- Tooling: Roboflow (annotation + dataset versioning)



\## Model

\- RF-DETR (Nano) trained on Roboflow

\- Input size: 512x512
